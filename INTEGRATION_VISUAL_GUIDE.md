## Color Coding & Visual Hierarchy

### Brain Gym (Purple/Pink)
- **Purpose:** Daily cognitive training
- **Icon:** Brain
- **Gradient:** Purple → Pink
- **Position:** Top of dashboard

### Cognitive Profile (Purple/Pink/Orange)
- **Purpose:** Self-understanding
- **Icon:** Brain
- **Gradient:** Purple → Pink → Orange
- **Position:** Middle of dashboard
- **NEW!** badge to attract attention

### Skill Builder (Teal/Cyan/Blue)
- **Purpose:** Targeted growth
- **Icon:** Target
- **Gradient:** Teal → Cyan → Blue
- **Position:** Below profile

### Career Matches (Blue/Purple)
- **Purpose:** Career guidance
- **Icon:** Briefcase
- **Gradient:** Blue → Purple (in buttons)
- **Access:** From profile card or full view

This creates a visual journey:
**Purple (understand)** → **Pink (discover)** → **Teal (grow)** → **Blue (apply)**

## Mobile Responsiveness

All views are fully responsive:
- Profile card: Stacks metrics on mobile
- Radar chart: Shrinks to fit
- Career cards: Single column on mobile
- Navigation: Touch-friendly 44px targets
- Buttons: Full-width on small screens

## Accessibility

- Semantic HTML structure
- ARIA labels on all buttons
- Keyboard navigation support
- Screen reader friendly
- Color + text labels (not color alone)
- High contrast ratios

## Performance

- Lazy loading: Profile only loads when shown
- Caching: Profile cached in state
- Debouncing: Auto-generation only after submission
- Progressive rendering: Dashboard loads first
- Error boundaries: Failures don't break app
