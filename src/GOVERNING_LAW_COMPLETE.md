## 🇬🇭 Ghana Children's Act 843 (1998)

### Core Information

**Full Title:** Children's Act, 1998 (Act 843)  
**Enacted:** December 30, 1998  
**Jurisdiction:** Republic of Ghana  
**Age of Majority:** 18 years  
**Parental Consent:** Required under 18  

---

### Key Sections Applied

#### Section 1: Definition of Child
```
"A person below the age of eighteen years"

✅ All users under 18 are minors
✅ Enhanced protections for all minors
✅ Parental consent required under 18
```

#### Section 4: Best Interest of the Child
```
"Best interest of the child shall be paramount"

✅ Child safety prioritized in design
✅ Features restricted when not beneficial
✅ Data practices minimize child risk
```

#### Section 6: Parental Duty
```
"Parents have duty to maintain, protect and provide"

✅ Parents must consent for child accounts
✅ Parents have oversight access
✅ Parents can terminate accounts
✅ Parents receive notifications
```

#### Section 11: Right to Privacy
```
"Right to privacy and protection of information"

✅ Strictest data minimization
✅ No behavioral tracking
✅ No advertising to children
✅ Data anonymization
```

#### Section 13: Right to Education
```
"Every child has right to education"

✅ Educational purpose design
✅ Learning support focus
✅ Age-appropriate content
✅ Academic development support
```

#### Section 28: Protection from Exploitation
```
"No exploitative labor or practices"

✅ No commercial data exploitation
✅ No data selling (ever)
✅ No commercial profiling
✅ Educational use only
```

## 🔧 Technical Implementation

### Detect Jurisdiction & Apply Laws

```typescript
import { 
  Jurisdiction, 
  getApplicableLaws, 
  getConsentAge,
  requiresParentalConsent
} from './types/governing-law-configs';

// Detect user location
const jurisdiction = detectJurisdiction(userIP);
// Returns: Jurisdiction.GHANA, Jurisdiction.US, etc.

// Get all applicable laws
const laws = getApplicableLaws(jurisdiction);
// Returns: Ghana laws + location-specific laws

// Get consent age (always uses higher protection)
const consentAge = getConsentAge(jurisdiction);
// Ghana: 18, US: 18 (Ghana stricter), EU: 18 (Ghana stricter)

// Check if parental consent needed
const needsConsent = requiresParentalConsent(userAge, jurisdiction);
if (needsConsent) {
  showParentalConsentFlow();
}
```

### Get Act 843 Protections

```typescript
import { getAct843Protections } from './types/governing-law-configs';

// Get Ghana Act 843 details
const act843 = getAct843Protections();

console.log(act843.actName);        // "Children's Act"
console.log(act843.actNumber);      // "Act 843"
console.log(act843.minorDefinition); // "Under 18 years"
console.log(act843.consentAge);      // 18

// Display relevant sections
act843.relevantSections.forEach(section => {
  console.log(`${section.section}: ${section.title}`);
  console.log(section.description);
});
```
