# 🖨️ Print Your Assessment Report - Quick Reference

## How to Print or Save as PDF

### **Method 1: Print Button** (Recommended)
1. Click the **"Print Report"** button at the top of your results page
2. Your browser's print dialog will open
3. Choose your destination:
   - **Save as PDF** → Creates a digital copy
   - **Select Printer** → Sends to physical printer
4. Click "Save" or "Print"

### **Method 2: Keyboard Shortcut**
- **Windows/Linux:** Press `Ctrl + P`
- **Mac:** Press `Cmd + P`
- Follow steps 3-4 above

## Print Settings Recommendations

### **For Digital Copy (PDF):**
```
Destination: Save as PDF
Pages: All
Color: Grayscale (saves file size)
Paper: A4 or Letter (doesn't matter for PDF)
```

### **For Physical Printing:**
```
Destination: Your printer
Pages: All
Color: Black & White (saves ink!)
Paper: A4 or Letter
Quality: Normal
```

## Tips & Tricks

💡 **Want to save ink?**
→ Select "Black & White" or "Grayscale" in print settings

💡 **Want to print just one section?**
→ Highlight the section, then select "Selection" in print dialog

💡 **Want to print multiple reports?**
→ Save each as PDF first, then print them all together

💡 **Want to email it?**
→ Save as PDF, then attach to email

💡 **Want to share to Google Drive/Dropbox?**
→ Save as PDF, then upload to your cloud storage

## Mobile Devices

### **iPhone/iPad:**
1. Tap "Print Report" button
2. Tap "Share" icon
3. Select "Save to Files" or "Print"
4. Choose location or printer

### **Android:**
1. Tap "Print Report" button
2. Choose "Save as PDF" or select printer
3. Tap download icon to save
4. Or select printer to print

## Troubleshooting

**Q: The print preview looks different from the screen**  
A: That's normal! The print version removes buttons and navigation to save space.

**Q: Why is everything black and white?**  
A: To save ink! If you need colors, select "Print backgrounds" in your print settings.

**Q: How do I get page numbers?**  
A: Most browsers add page numbers automatically in the footer.

**Q: Can I print just the scores?**  
A: Yes! Highlight the scores section, then choose "Selection" in the print dialog.

**Q: The charts look different**  
A: They're optimized for black & white printing so they're readable without color.
