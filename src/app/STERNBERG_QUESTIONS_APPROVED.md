## 🌟 Response Format

All 100 questions use the **5-point Likert scale**:

| Value | Label | Description |
|-------|-------|-------------|
| 1 | Strongly Disagree | This statement does NOT describe me |
| 2 | Disagree | This statement describes me a little |
| 3 | Neutral | I'm unsure or in between |
| 4 | Agree | This statement describes me well |
| 5 | Strongly Agree | This statement describes me VERY well |

## 💯 Scoring System

### Individual Style Scores:
```
Analytical Score = Sum of Likert values (Q1-34)
  Max: 170 (34 questions × 5)
  Min: 34 (34 questions × 1)

Creative Score = Sum of Likert values (Q35-67)
  Max: 165 (33 questions × 5)
  Min: 33 (33 questions × 1)

Practical Score = Sum of Likert values (Q68-100)
  Max: 165 (33 questions × 5)
  Min: 33 (33 questions × 1)
```

### Percentage Calculation:
```
Total Score = Analytical + Creative + Practical

Analytical % = (Analytical Score / Total Score) × 100
Creative % = (Creative Score / Total Score) × 100
Practical % = (Practical Score / Total Score) × 100
```

### Dominant Style:
```
Dominant Style = max(Analytical %, Creative %, Practical %)
```
