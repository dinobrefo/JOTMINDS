# ⚡ Kids Mode - Quick Start Guide

## 🚀 **How to Use Kids Mode**

### **For Users (Ages 6-10)**

#### **Automatic Activation**
1. **Create Account** or **Login** with your birthdate
2. If you're **6-10 years old**, Kids Mode activates automatically
3. Meet **Jot the Robot** - your learning companion!
4. Start taking quizzes with big buttons and fun animations

#### **Manual Access**
1. Navigate to landing page
2. Click **"Try Kids Mode Demo"**
3. Explore without login required

---

### **For Parents**

#### **Setting Up Kids Mode**
1. Create account for your child (age 6-10)
2. Kids Mode activates based on birthdate
3. Set a **4-digit Parent PIN** for protection
4. Child can now safely use the platform

#### **Monitoring Progress**
1. Login to **Parent Dashboard**
2. View linked child accounts
3. See assessment results
4. Track progress and achievements

#### **Exiting Kids Mode**
1. Click the **Exit button** (requires Parent PIN)
2. Re-enter adult interface
3. Access full dashboard features

---

### **For Teachers**

#### **Classroom Setup**
1. Create student accounts (ages 6-10)
2. Kids Mode auto-activates for eligible students
3. Students take assessments independently
4. Review results in Teacher Dashboard

#### **Best Practices**
- ✅ Enable audio narration for auditory learners
- ✅ Allow students to work at their own pace
- ✅ Celebrate achievements with the class
- ✅ Use results to guide instruction

## 🔧 **Troubleshooting**

### **Audio Not Working?**
1. Check browser audio permissions
2. Unmute device
3. Click audio button manually (some browsers require user interaction)
4. Try different browser

### **Kids Mode Not Activating?**
1. Verify age is 6-10 years old
2. Check birthdate in account settings
3. Try manual demo mode
4. Contact support if issue persists

### **Can't Exit Kids Mode?**
1. Click Exit button in top corner
2. Enter your 4-digit Parent PIN
3. If forgotten, use password reset
4. Contact support for assistance

### **Quiz Not Saving?**
1. Check internet connection
2. Complete all questions
3. Wait for "saving" confirmation
4. Refresh if needed

## 🎯 **Tips for Success**

### **For Children**
- ✅ Pick the answer that feels right to YOU
- ✅ There are no wrong answers
- ✅ Take your time
- ✅ Click the speaker to hear questions again
- ✅ Have fun with Jot!

### **For Parents**
- ✅ Let children work independently
- ✅ Don't influence their choices
- ✅ Celebrate completion, not "right" answers
- ✅ Use results to support learning
- ✅ Review together afterward

### **For Teachers**
- ✅ Brief students on purpose (5 min)
- ✅ Ensure quiet environment
- ✅ Allow 15-20 minutes per quiz
- ✅ Provide headphones if available
- ✅ Use results for differentiation
