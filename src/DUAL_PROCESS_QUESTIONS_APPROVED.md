## 🌟 Response Format

All 100 questions use the **5-point Likert scale**:

| Value | Label | Description |
|-------|-------|-------------|
| 1 | Strongly Disagree | This statement does NOT describe how I decide |
| 2 | Disagree | This statement describes me a little |
| 3 | Neutral | I'm unsure or in between |
| 4 | Agree | This statement describes how I decide well |
| 5 | Strongly Agree | This statement describes how I decide VERY well |

## 💯 Scoring System

### Individual Style Scores:
```
Intuitive Score = Sum of Likert values (Q1-50)
  Max: 250 (50 questions × 5)
  Min: 50 (50 questions × 1)

Reflective Score = Sum of Likert values (Q51-100)
  Max: 250 (50 questions × 5)
  Min: 50 (50 questions × 1)
```

### Percentage Calculation:
```
Total Score = Intuitive + Reflective

Intuitive % = (Intuitive Score / Total Score) × 100
Reflective % = (Reflective Score / Total Score) × 100
```

### Dominant Style:
```
Dominant Style = max(Intuitive %, Reflective %)
```

## 🔍 Theoretical Alignment

### Dual-Process Theory (Kahneman, 2011):

**System 1 - Intuitive (Q1-50):**
- Fast, automatic, effortless
- Emotional and experiential
- Pattern-based, associative
- Operates unconsciously
- Good for familiar situations
- Prone to cognitive biases

**System 2 - Reflective (Q51-100):**
- Slow, controlled, effortful
- Logical and analytical
- Rule-based, sequential
- Operates consciously
- Good for novel problems
- Can override System 1

**Key Insight:** Most people use BOTH systems. The questions measure preference, not ability.

## 🔧 Practical Applications

### For Students (Ages 11-14):

**High Intuitive Scorers:**
- ✅ Strengths: Quick reactions, confident in emergencies, trusts experience
- ⚠️ Watch for: Rushing important decisions, missing details, impulsive choices
- 💡 Tips: Slow down for tests/homework, check work before submitting

**High Reflective Scorers:**
- ✅ Strengths: Careful analysis, considers consequences, avoids mistakes
- ⚠️ Watch for: Overthinking, analysis paralysis, slow task completion
- 💡 Tips: Trust your first instinct sometimes, set time limits for decisions

**Balanced Scorers:**
- ✅ Strengths: Flexible decision-making, adapts to situation
- 💡 Tips: Use System 1 for simple/familiar, System 2 for complex/novel

## 🎯 Real-World Decision Scenarios

### When to Use Each System:

**Use Intuitive (System 1) for:**
- Simple, familiar decisions (what to eat, what to wear)
- Time-pressured situations (emergencies, quick responses)
- Social interactions (reading emotions, quick replies)
- Pattern recognition (familiar problems)
- Sports and games (fast reactions needed)

**Use Reflective (System 2) for:**
- Complex decisions (choosing schools, long-term plans)
- Unfamiliar situations (new problems)
- Important consequences (exams, big projects)
- Mathematical reasoning (calculations, logic)
- Critical thinking tasks (analyzing arguments)
